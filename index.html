Comandos usados em Git

Abra o Git bash

Para criar um repositorio crie uma pasta em algum local em sua maquina e clique com botão direito e selecione Git bash here

ou

Abra o Git Bash e com o comando $ cd "pasta" vai escrevendo $ cd até chegar na pasta destinada, caso entre na errada use o comando $ cd.. com isso voce ira retroceder uma pasta e caso entre em alguma mas nao se recorda em qual pasta se encontra, digite o comando $ cd e precione a tecla tab do teclado duas vezes que ira aparecer as sub-pastas que se encontram no local atual

$ git init : Ira mapear seus arquivos para fazer o versionamento do projeto

$ code . : ira abrir o VsCode para criar o primeiro arquivo

Crie tambem um repositorio no seu repositorio remoto. Copie o link do repositorio criado que sera necessario para o proximo passo

$ git remote add origin "link-do-repositorio-criado"

: Depois disso faça um push para enviar todos os arquivos do projeto para o repositorio remoto.

$ git checkout "nome-da-branch" : usada para selecionar a branch em qual deseja trabalhar

$ git checkout -b "nome-da-branch" : Criar uma nova ramificacao para nao necessitar alterar diretamente na master.

Apos modificar o codigo e deseja fazer o commit para o repositorio remoto.

$ git status : Ira verificar oque foi modificado no seu projeto.

$ git add . : Ira selecionar tudo que esta modificado para o proximo commit.

$ git add "nome-especifico" : pode ser usado para adicionar algo modificado aparte

$ git commit -m "breve-descricao": Faz o commit da branch que voce esta gerenciando no momento

$ git push origin "branth-atual": Envia sua branch modificada para o repositorio local

Selecione a branch principal para fazer a merge

$ git merge --no-ff "branch-secundaria": Faz o merge, mesca a branch-secuntaria com a master ou a main

se acontecer algum conflito, corrija o problema e depois verfique o status da branch e apos isso faça update para o repositorio remoto

Para fazer a exclusão de uma branch é necessario usar o comando $ git branch -d "nome-da-branch" isso ira remover a branch localmente

para removela remotamente do meu repositorio é necessario usar $ git push origin --delete "nome-da-branch" isso ira fazer a exclusão da branch remota
