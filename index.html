Comandos usados em Git

Abra o git bash

$ git checkout "nome-da-branch" : usada para selecionar a branch em qual deseja trabalhar

$ git checkout -b "nome-da-branch" : Criar uma nova ramificacao para nao necessitar alterar diretamente na master.

Apos modificar o codigo e deseja fazer o commit para o repositorio remoto.

$ git status : Ira verificar oque foi modificado no seu projeto.

$ git add . : Ira selecionar tudo que esta modificado para o proximo commit.

$ git add "nome-especifico" : pode ser usado para adicionar algo modificado aparte
